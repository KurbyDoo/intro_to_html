So what is HTML?

HTML or Hypertext Markup Language was designed to define the structure and meaning of a web pages content.
Since being created in 1993, the language has gone through 5 different versions and is now commonly used with CSS or Cascading Style Sheets to design the presentation of web pages, and Javascript to add functionallity.

The Basics

To begin writing HTML, create a file using a text editor or an IDE (Integrated Developemnt Environment) ending in .html
Then write "<!DOCTYPE html>" at the top of the file.

HTML is strucutred using Elements which contain Markup Tags.
They are what define and structure web pages.
Each element is defined using a starting tag <> and must be closed using and ending tag </>.
There are many different types of tags including

  • <h1> Header tags
  • <p> Paragraph tags
  • <div> Division tags
  • <br> Line break tags
  • <a> Anchor tags

And more

HTML elements can also be nested within each other. For example the <body> tag, which represents the main site is nested within the <html> element.

Why use formatting?

Text formatting is used to improve the readability of a web page. For example, the use of different sized headers can help readers easily locate different sections of the page and quickly find the information they need. Formatting can also be used stylistically to give a web page a certain aesthetic design for its readers

All About <Elements>

How do tags work?

HTML markup tags define how web browsers will format and display its content. They consist of an opening tag, inner content, and a closing tag. HTML markup tags start and end with the <> brackets and are preferably written in lowercase for readability and compatibility with stricter document types such as XHTML.

Empty tags

Some tags do not include content and do not need a closing tag.
These tags include:
<br> which adds a line break in between elements.
<hr> which adds a horizontal line across the page.

What are attributes?

Attributues are additional parameters HTML elements can be given in order to specify more information about how the element functions. They are specifed in the opening tag of an element and are structured in name/value pairs.

Here are some examples of attributes:

The "href" attribute

<a href="https://google.com"> This is a link to google.com </a>

The "style" attribute

<p style="color: red;"> This makes the text red! </p>

The "title" attribute

<p title="This will display as a tooltop!"> Hover over me to display a tooltip! </p>

The "target" attribute

<a href="https://www.target.com/" target="_blank">
This will open on a new tab </a>

Important Tips!

How to explain your code

Comments are a usefull way to explain your code! They can be used to explain what your code does to yourself and other readers. In HTML, comments are marked with the comments tag "<!-- -->". In CSS, comments are marked with "/* */".

Here is an example:

<!-- this paragraph contains important information -->

<p class="important__paragraph">

Important information!

</p>

/* this class is for important information */

.important__paragraph {

font-size: 2rem;

}

Styling

Styles are useful to change the way a webpage looks!

They can turn blank pages into more visually appealing sites. Styling can be done with the <style> tag in HTML, but a better more common method to use a stylesheet. Stylesheets seperate styles from your HTML document for a more organised experience. To do this, simply create a new file ending with .css and link the stylesheet to your html document.
